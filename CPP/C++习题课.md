#### 最大回文子串

###### 动态规划

基本的思想是每个回文串去头去尾仍然是回文串，由此从短的串推出长的子串是否回文，并记录每个子串回文状态。

```c++
dp[i][j] = s[i] == s[j] && dp[i+1][j-1]==1;
//dp[i][j]是以i开头，j结尾的子串是否是回文串
```

初始状态是长度为1，2的串，根据串长度、开头位置双循环，计算出所有回文子串

同时，每次维护最长的长度。（因为每次计算出的长度都是最长的）
